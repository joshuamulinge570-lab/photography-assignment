PROJECT NAME:Joshua's Photography Website
AUTHOR NAME:Joshua Mulinge

DESCRIPTION OF PROJECT
It is a simple web project built using HTML and CSS.
It shows different categories of photography including portrait,landscape,wildlife
and street photography.Each image is interactive and clickable allowing users to 
view a larger section in a new tab.

PROJECT SETUP INSTRUCTIONS
1.Clone the repository

2.Navigate into the project folder
  cd photography-assignment
3.Open the project
  double click index.html
4.Ensure all image files are in the same folder as index.html
